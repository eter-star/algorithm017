01.Two Sum java代码

这道题目的one-pass hashmap的解法有一个前提是，数组当中存在唯一解；如果存在多种解，应该把所有的解保存到一个linkedlist当中
  1. class Solution {
  2.  public int[] twoSum(int[] nums, int target) {
  3.  //One-pass hashmap on premise that there exist only one solution
  4.  Map<Integer, Integer> map = new HashMap<Integer, Integer>();
  5.  for(int i =0; i < nums.length; i++){
  6.  int completment = target - nums[i];
  7.  if(map.containsKey(completment)){
  8.  return new int[]{i, map.get(completment)};
  9.  }
  10.  map.put(nums[i], i);
  11.  }
  12.  return null;
  13.  }
  14. }

21. merge two lists java代码

采用递归方法求解，也可以采用iteration的方法求解
  1. /**
  2.  * Definition for singly-linked list.
  3.  * public class ListNode {
  4.  * int val;
  5.  * ListNode next;
  6.  * ListNode() {}
  7.  * ListNode(int val) { this.val = val; }
  8.  * ListNode(int val, ListNode next) { this.val = val; this.next = next; }
  9.  * }
  10.  */
  11. class Solution {
  12.  public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
  13.  if(l1 == null) return l2;
  14.  if(l2 == null) return l1;
  15.  ListNode head = null;
  16.  if(l1.val < l2.val) head = new ListNode(l1.val, mergeTwoLists(l1.next, l2));
  17.  else head = new ListNode(l2.val, mergeTwoLists(l1, l2.next));
  18.  return head;
  19.  }
  20. }

24. swap pairs java代码

iteration方法，需要建立一个dummy node
  1. /**
  2.  * Definition for singly-linked list.
  3.  * public class ListNode {
  4.  * int val;
  5.  * ListNode next;
  6.  * ListNode() {}
  7.  * ListNode(int val) { this.val = val; }
  8.  * ListNode(int val, ListNode next) { this.val = val; this.next = next; }
  9.  * }
  10.  */
  11. class Solution {
  12.  public ListNode swapPairs(ListNode head) {
  13.  if(head == null || head.next == null) return head;
  14.  ListNode dummy = new ListNode(-1);
  15.  dummy.next = head;
  16.  ListNode prevNode = dummy;
  17.  ListNode firstNode = null;
  18.  ListNode secondNode = null;
  19.  while(head!=null && head.next !=null){
  20.  firstNode = head;
  21.  secondNode = head.next;
  22.  prevNode.next = secondNode;
  23.  firstNode.next = secondNode.next;
  24.  secondNode.next = firstNode;
  25.  prevNode = firstNode;
  26.  head = firstNode.next;
  27.  }
  28.  return dummy.next;
  29.  }
  30. }
递归方法
  1. class Solution {
  2.  public ListNode swapPairs(ListNode head) {

  5.  ListNode newHead = head.next;
  6.  newHead.next = head;
  7.  head.next = p;

  9.  }

  1. class Solution {
  2.  public int[] plusOne(int[] digits) {
  3.  int n = digits.length;
  4.  int bitIncrease = (digits[n - 1] + 1)/10 ;
  5.  digits[n - 1] = (digits[n- 1] + 1) % 10;
  6.  for(int i = n - 2; i >= 0 ; i--){
  7.  digits[i] += bitIncrease;
  8.  bitIncrease = digits[i] / 10;
  9.  digits[i] %= 10;
  10.  }
  11.  if(bitIncrease == 1){
  12.  int[] arr = new int[n + 1];
  13.  for(int i = 1; i < n+1; i++){
  14.  arr[i] = digits[i-1];
  15.  }
  16.  arr[0] = 1;
  17.  return arr;
  18.  }else{
  19.  return digits;
  20.  }
  21.  }
  22. }

77 climb stairs java代码

使用递归方法的Fibonnaci数列求解，算法时间复杂度O（n），利用aux函数，将两个参数Fibo-prev, Fibo_prevprev传入函数参数
  1. class Solution {
  2.  public int climbStairs(int n) {
  3.  if(n == 1) return 1;
  4.  if(n == 2) return 2;
  5.  return climbStairsAux(n, 2 , 1); 
  6.  }
  7.  public int climbStairsAux(int n, int nMinusOne, int nMinusTwo){
  8.  if(n < 3) return nMinusOne;
  9.  return climbStairsAux(n-1, nMinusOne + nMinusTwo , nMinusOne);
  10.  }
  11. }

189 rotate array java代码

涉及到数组长度与k值的最小公约数gcd的计算。此外，leetcode上给出了一种全局计算count的循环方法。
corner case 是处理过的 k==0 的情况
  1. class Solution {
  2.  public void rotate(int[] nums, int k) {
  3.  k = k% nums.length;
  4.  if(k == 0 ) return;
  5.  int j = 0;
  6.  int gcd = gcd(nums.length, k);
  7.  while(j++ < gcd){
  8.  int prev = nums[j];
  9.  int count = 0;
  10.  for(int i = j; count++ < nums.length/gcd; i+=k){
  11.  int tmp = nums[(i+k) % nums.length];
  12.  nums[(i+k) % nums.length] = prev;
  13.  prev = tmp;
  14.  }
  15.  }
  17.  }
  19.  public int gcd(int a, int b){
  20.  while(a != b){
  21.  if(a > b){ a = a-b; }
  22.  else{ b = b-a;}
  23.  }
  24.  return a;
  25.  }
  26. }
